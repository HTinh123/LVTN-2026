# Luận Văn Tốt Nghiệp - Xây dựng hệ thống quản lý điểm rèn luyện cho sinh viên STU

## Thông tin sinh viên

### Họ \& tên: Phạm Lê Hướng Tinh

### MSSV: DH52201573

### Lớp: D22\_TH14



## Hệ thống sử dụng mô hình Monorepo để quản lý cả Frontend và Backend trong một kho lưu trữ duy nhất.

## Công nghệ sử dụng

* **Frontend:** HTML5, CSS3, JavaScript, React (Fetch API).
* **Backend:** Node.js, Express.js.
* **Database:** MySQL.
